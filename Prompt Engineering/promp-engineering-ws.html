Before & After:
See the Difference

The same task, two different prompts. Watch how specificity, context, and structure transform AI output from generic to gold.

Example 1 — Writing a professional email
✕ Vague prompt
Prompt
Write me an email to my boss about being late.
🤖 AI Response
Subject: Being Late

Dear Boss,

I am writing to inform you that I was late today. I apologize for the inconvenience. It won't happen again.

Sincerely, [Your Name]
✓ Engineered prompt
Prompt
Write a professional but warm email to my manager Sarah. I was 20 minutes late to the team standup today because of a traffic accident on the highway. I want to apologize sincerely, briefly explain what happened, and propose that I'll dial in from my phone if I'm ever stuck in traffic again. Keep it under 100 words. Tone: respectful but not overly formal — we have a good working relationship.
🤖 AI Response
Subject: Sorry about this morning

Hi Sarah,

I owe you an apology for missing the first 20 minutes of standup. There was a multi-car accident on the I-90 that brought traffic to a standstill — no way around it, unfortunately.

I know the standup sets the pace for the day, so going forward, if I'm ever stuck like that again, I'll dial in from my phone rather than show up late.

Thanks for understanding. I'll make sure the team has my updates first thing tomorrow.

Best, [Name]
What changed: Adding the recipient's name, the specific situation, a proposed solution, word limit, and tone guidance turns a generic template into a ready-to-send email.

Prompt

🤖 AI Response

Example 3 — Coding Help
✕ Vague prompt
Prompt
Write a Python function to process data.
🤖 AI Response
def process_data(data):
  result = []
  for item in data:
    result.append(item)
  return result

(…useless function that does nothing meaningful)
✓ Engineered prompt
Prompt
Write a Python function called `clean_sales_csv` that:
- Takes a file path to a CSV with columns: date, product, revenue, region
- Removes rows where revenue is negative or null
- Converts date strings ("MM/DD/YYYY") to datetime objects
- Returns a pandas DataFrame sorted by date descending
- Include type hints and a docstring
- Handle the case where the file doesn't exist (raise FileNotFoundError with a helpful message)
🤖 AI Response
import pandas as pd
from pathlib import Path

def clean_sales_csv(file_path: str) -> pd.DataFrame:
  """Clean and validate sales CSV data..."""
  path = Path(file_path)
  if not path.exists():
    raise FileNotFoundError(
      f"Sales file not found: {path}")
  df = pd.read_csv(path)
  df = df.dropna(subset=["revenue"])
  df = df[df["revenue"] >= 0]
  df["date"] = pd.to_datetime(
    df["date"], format="%m/%d/%Y")
  return df.sort_values("date",
    ascending=False).reset_index(drop=True)
What changed: Specifying the function name, input format, exact columns, edge cases, and code quality requirements (type hints, docstrings) yields production-ready code instead of a placeholder.

How LLMs Actually Think:
Sampling & Hallucination

LLMs don't "know" answers — they predict the most likely next token. Understanding this reveals why they sometimes get things confidently wrong.

How many r's are there in the word "garlic"?
🧑
🤖
The word "garlic" has 1 letter r.
UNDER THE HOOD — THE MODEL SEES TOKENS:
PROBABILITY DISTRIBUTION FOR THE ANSWER:
1
80%
0
18%
2
1.9%
The correct answer is 1, but ~18% of the time the model says 0. The model picks from a probability distribution — it doesn't actually "count" letters. This is why you should ask it to spell out the word letter-by-letter first.

Module 05

The 10 Pillars of
Great Prompts

Each pillar includes a concrete before → after example you can learn from immediately.

01Clarity
Be precise about what you want. Vague requests get vague responses.
Tell me about marketing → Explain 3 digital marketing strategies effective for small e-commerce businesses selling handmade jewelry
02Context
Provide background info. AI performs better with situation, audience, and purpose.
Write a welcome message → I run a yoga studio for beginners aged 40+. Write a welcome email for someone who just signed up for their first class and might be nervous
03Format
Specify structure: essay, bullets, table, JSON, numbered steps, etc.
Compare these databases → Compare PostgreSQL vs MongoDB in a table with columns: Use Case, Scalability, Learning Curve, Cost. Rate each 1-5
04Tone
Indicate voice: formal, casual, technical, ELI5, sarcastic, academic.
Explain blockchain → Explain blockchain to my 70-year-old mother who barely uses email. Use analogies she'd understand, like a shared family recipe book
05Examples
Show input → output pairs to teach the AI your expected format.
Convert these to slug format:
"Hello World" → "hello-world"
"My Blog Post!" → "my-blog-post"
Now convert: "AI & Machine Learning 101"
06Length
State desired length explicitly — don't leave it to chance.
Summarize this article → Summarize this article in exactly 3 bullet points, each under 20 words. Then add a one-sentence "so what?" takeaway
07Decomposition
Break complex requests into steps. Ask AI to think before answering.
Plan my product launch → Think step by step: 1) Identify the 3 biggest risks for launching a B2B SaaS in Q3, 2) For each risk, suggest a mitigation, 3) Then create a 4-week timeline
08Constraints
Set boundaries: what to include, exclude, avoid, or require.
Write a diet plan → Create a 5-day meal plan. Constraints: vegetarian, under 1800 cal/day, no soy (allergy), budget max €8/meal, prep time under 30 min
09Role
Assign a persona: teacher, skeptic, editor, domain expert.
Review my business plan → Act as a venture capitalist who has seen 500 pitches this year. Review my business plan. Be brutally honest about what wouldn't survive due diligence
10Iteration
Build feedback loops. Tell AI when to ask questions vs. assume.
Help me with my resume → Before rewriting my resume, ask me 5 questions about the target role, my top achievements, and what tone I want. Then draft based on my answers

Module 07

Advanced Prompt
Patterns

Battle-tested patterns used by professionals.

Reasoning

Chain-of-Thought

Ask AI to show reasoning step-by-step. Dramatically improves accuracy.

"A store has 3 boxes. Each box has
4 bags. Each bag has 5 marbles.
How many marbles total?

Think step by step before answering."
Quality

Few-Shot Prompting

Provide 2–3 input → output examples. The AI pattern-matches.

// Teach the format:
Email: "loved the product!"
Category: Praise
Priority: Low

Email: "my order never arrived"
Category: Complaint
Priority: High

// Now classify this one:
Email: "can I change my address?"
Perspective

Multi-Persona Debate

Multiple viewpoints force deeper, more balanced analysis.

"Should we switch to microservices?

Argue from 3 perspectives:
1. The CTO (technical debt)
2. The CFO (cost & risk)
3. A senior dev (daily impact)

Then give your recommendation."
Refinement

Iterative Sculpting

Draft → Critique → Rewrite. Each pass sharpens the output.

You: "Write a product description
  for our new headphones."
AI: [drafts v1]
You: "Now critique this. What's
  weak? What would a copywriter
  change?"
AI: [identifies 4 issues]
You: "Rewrite fixing all 4 issues."

Prompt Recipes

Ready-to-use templates you can adapt to your own work in seconds. Replace the highlighted placeholders with your details.

📧

The Perfect Email Drafter

Works for any professional email — apologies, requests, follow-ups, pitches.
Write a [type: apology/request/follow-up] email to [recipient + relationship]. Situation: [what happened / what you need] Tone: [warm/formal/direct/casual] Length: Under [word count] words. Must include: [specific point or call-to-action] Must avoid: [anything to steer clear of]
🔍

The Expert Analyst

Turns AI into a domain expert who delivers structured, actionable analysis.
Act as a senior [domain: financial/marketing/legal] analyst with 15 years of experience. I need you to analyze: [topic or data] Context: I'm a [your role] at a [company type]. This analysis is for [audience: my CEO / board / team]. Structure your response as: 1. Executive summary (3 sentences) 2. Key findings (top 3-5, with data) 3. Risks I should flag 4. Recommended next steps Be specific. Use numbers where possible. If you're uncertain about anything, say so.
💻

The Code Generator

Gets production-quality code instead of toy examples.
Write a [language] function called `[function_name]` that: Input: [describe input type and format] Output: [describe expected return] Requirements: - [requirement 1] - [requirement 2] - Handle edge cases: [list them] - Include type hints and docstring - Add 3 unit test examples If anything is ambiguous, ask me before coding.
🧠

The Decision Helper

For when you're stuck between options and need structured thinking.
I'm deciding between: [Option A] vs [Option B] (vs [Option C if applicable]). Context: [your situation, constraints, goals] My priorities (ranked): 1. [most important factor] 2. [second factor] 3. [third factor] For each option, give me: - Pros (top 3) - Cons (top 3) - The one thing most people overlook Then make a recommendation based on my stated priorities. Play devil's advocate against your own recommendation.
